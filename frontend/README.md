# Frontend - Hệ Thống Quản Lý Gia Phả

React frontend cho ứng dụng quản lý gia phả

## Cài Đặt

```bash
npm install
```

## Chạy

### Development
```bash
npm start
```

Ứng dụng sẽ mở tại http://localhost:3000

### Build Production
```bash
npm run build
```

## Tính Năng

- Đăng ký / Đăng nhập
- Xem danh sách dòng họ
- Xem cây gia phả trực quan
- Quản lý thành viên
- Thống kê

## Công Nghệ

- React 18
- React Router v6
- Axios
- react-d3-tree (visualization)
- React Toastify
